# QA Portfolio

Portfólio de testes manuais em construção, com projetos práticos de Bug Hunt realizados em aplicações reais e open source. Cada projeto inclui casos de teste documentados, execução real e bugs encontrados com evidências.

Estou em transição de carreira para QA Analyst, estudando testes de software de forma estruturada e documentando cada etapa prática aqui. Os projetos abaixo refletem testes feitos por mim, com raciocínio próprio sobre o que testar e por quê — não conteúdo gerado automaticamente.

## Habilidades praticadas

Testes funcionais e exploratórios, documentação de casos de teste, identificação e relato de defeitos, controle de versão com Git e GitHub.

## Sobre o processo

Cada projeto segue a mesma estrutura: explorar a aplicação, escrever casos de teste cobrindo cenários positivos e negativos, executar e registrar resultados reais, e documentar qualquer defeito encontrado com passos de reprodução e evidência visual.
